# 入学説明会 用品申込

入学説明会で、iPad からその場で制服・ジャージ・作業着などを申し込むためのアプリ。

- ファイルは `index.html` 1つだけ
- データは iPad の中（localStorage）にだけ保存されます。外部サーバーには一切送信しません
- 販売元は いさごだ / 田丸スポーツ / 学校 の3系統
- 支払いは後日、それぞれの販売元で商品を受け取る際

## 使い方

1. **設定タブ** で端末番号（1号機・2号機…）を iPad ごとに変える
2. **設定タブ** で商品マスタ（品名・サイズ・単価・販売元）を登録
3. **受付タブ** で保護者に入力してもらう
4. 終了後、各 iPad の「データを書き出す」→ AirDrop で先生の端末へ
5. **一覧・出力タブ** で読み込み → 申込用紙・集計表を印刷、CSV を Teams へ

## 注意

- 端末番号を iPad ごとに変えないと受付番号が重複します
- 説明会が終わったら、印刷とバックアップを済ませてから「受付データを全部消す」を実行してください
